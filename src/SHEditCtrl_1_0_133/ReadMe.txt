SHEditCtrl dll module

본 모듈은 상업적 목적으로 사용을 금합니다.
또한 사용상 발생하는 문제에 있어서 개발자가 책임지지 않습니다.
API 문서는 따로 만들지 않았으며, SHEditCtrl.h 함수 원형과 설명을 들어있습니다.

각 파일 설명
	SHEditCtrl.h	- SHEditCtrl header file
	
	SHEditCtrl.dll	- SHEditCtrl 구현 dll
	SHEditCtrl.lib	- SHEditCtrl lib
	SHEditCtrlD.dll	- SHEditCtrl 구현 dll(Debug mode)
	SHEditCtrlD.lib	- SHEditCtrl lib(Debug mode)
